This folder contains the files necessary to compare the performances of Pixel and Mothra across manuscripts with different notations. The manuscripts are:
- [CH-Fco Ms. 2](https://cantusdatabase.org/source/123672)
- [CH-P 18](https://cantusdatabase.org/source/638308)
- [D-KNd 1161](https://cantusdatabase.org/source/601861)
- [D-WIl Hs. 2](https://cantusdatabase.org/source/588308)
- [F-Pn Latin 15181](https://cantusdatabase.org/source/123631)

Two folios are annotated per manuscript. Each folder's folio contains:
- The folio image;
- The three layers as separated in Pixel;
    - Layer 1 is neumes
    - Layer 2 is staves
    - Layer 3 is text
- The json and YOLO outputs of the layer annotation in Mothra.

As of 11.02.26, this work is ongoing. You can track the progress via this spreadsheet: https://docs.google.com/spreadsheets/d/1K_5yCKC0514RhpI1Lw0fEmRgGbwWakga1OaEQ3wt0qQ/edit?gid=0#gid=0
